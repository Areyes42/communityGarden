# App Structure Outline

## Client: HTML, CSS, JavaScript

## Server: Flask

## Database: MongoDB Atlas

## Data
- User
    - Username: str
    - Password: str
    - Plant: Plant

- Plant
    - Growth Stage: int
    - Sunlight: int
    - Water: int

## Functionality
- User Registration
- User Authentication
- Water Plant
    - Water gained through "good deeds"
- Sunlight
    - Sunlight gained through app interaction
